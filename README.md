# pyDSP
Simple DSP lib just for fun and better algorithms understanding.

It uses python3 and numpy, desired to work with complex numbers.

correlation.py
- cross correlation
- auto correlation

svdj.py
- Jacoby SVD

fir.py
- 3 different implementations of universal FIR
- FIR 6 TAP
- FIR 7 TAP
- FIR 8 TAP
- FIR 12 TAP

fir_up2.py
- FIR UP 2 with optimizations
